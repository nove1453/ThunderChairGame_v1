# 電気イスゲーム

2人で1台のスマートフォンを交互に渡して遊ぶ、静的Webアプリ版の電気イスゲームです。GitHub Pagesにそのまま配置して動作します。

## ゲームルール

- 1〜12番のイスを使います。
- 各ターンで、片方のプレイヤーが残っているイスから1脚に電流をセットします。
- 端末を渡したあと、もう片方のプレイヤーが座るイスを選びます。
- 電流を回避した場合、選んだイス番号と同じ点数を獲得し、そのイスは盤面から除外されます。
- 電流を受けた場合、現在得点が0点になり、電流回数が1回増えます。そのイスは盤面に残ります。
- 40点以上に到達、相手が電流3回、または回避によって残り1脚になった時点で終了します。

## ファイル構成

```text
/
├── index.html
├── style.css
├── app.js
├── assets/
│   └── electric-shock.mp3
└── README.md
```

`assets/electric-shock.mp3` は任意です。ファイルがなくてもアプリは停止しません。

## ローカルでの起動方法

このフォルダをブラウザで開くだけでも動作します。ローカルサーバーで確認する場合は、以下を実行してください。

```bash
python3 -m http.server 8000
```

その後、ブラウザで `http://localhost:8000/` を開きます。

## GitHub Pagesでの公開方法

1. このフォルダのファイルをGitHubリポジトリに配置します。
2. GitHubの `Settings` から `Pages` を開きます。
3. 公開元に `main` ブランチのルート、または配置したブランチを指定します。
4. 表示されたURLにアクセスします。

## 効果音ファイル

電撃音を使う場合は、MP3ファイルを次のパスに置いてください。

```text
assets/electric-shock.mp3
```

音量はJavaScript側で0.7に設定しています。iPhone Safariではユーザー操作を起点にした音声再生のみ許可されるため、電流判定時の「確定」操作から再生を試みます。再生に失敗してもゲーム進行は止まりません。

## Safariでの注意

- iPhone Safariの音声制限により、初回再生が端末設定やブラウザ状態によって失敗する場合があります。
- 画面点滅が苦手な人向けに「画面点滅を減らす」設定を用意しています。
- OSの視差効果を減らす設定が有効な場合も、揺れと点滅を弱めます。
